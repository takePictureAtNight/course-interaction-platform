# 答卷列表

## 请求信息

### 请求地址

```
http://192.168.31.82:8082/courseresourse/surveydata/list?limit=5&page=2
```

### 请求方法

```
GET
```

## 入参

### 入参示例 (RequestParam)

```
//page:2

```

| -------- | -------- | -------- | -------- | -------- |
| limit     | **Integer**     | 否  | 页面大小  |   |
| page     | **Integer**     | 否  | 当前页数  |   |
